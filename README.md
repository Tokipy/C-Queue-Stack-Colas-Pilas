# Hospital Queue System - C Project

This project is a simulation of a hospital patient management system developed in C, using dynamic memory and fundamental data structures: queues and stacks. The system uses dynamic memory allocation (malloc) to manage said data structures efficiently.

## Description

The program simulates the arrival of patients in a hospital and assigns them to different queues based on their level of urgency.

Key features include:
- Patient classification by priority level
- Management of multiple queues (one for each urgency level)
- Processing of patients based on priority
- Storage of attended patients in a stack

## Technologies and Concepts

- C Programming Language
- Dynamic memory management (malloc)
- Queues (FIFO)
- Stacks (LIFO)
- Pointers and structured data

## Purpose

This project was developed as part of a university evaluation, with the goal of applying data structures and memory management concepts in a practical scenario. It was completed under time constraints, demonstrating problem-solving skills and the ability to implement efficient solutions under pressure. The project achieved the highest possible grade (10/10), reflecting correctness, efficiency, and proper implementation of the required concepts.
